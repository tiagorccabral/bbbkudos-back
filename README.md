# Projeto BBB Kudos

----

### Versões das bibliotecas

Ruby: 2.6.3

Rails: 6.0.3.1

### Modelo do BD
![GitHub Logo](db.png)

### Instalação do projeto
```
$ git clone https://github.com/tiagorccabral/bbbkudos-back.git
$ cd bbbkudos-back
$ bundle install
$ rails db:create db:migrate db:seed
```

### Rodando o projeto

```
$ rails s
```
